Перший розділ

HTML (HyperText Markup Language) є мовою розмітки гіпертексту, яку використовують переважно для створення документів у мережі інтернет.

HTML почав свій шлях на початку 90-х років як примітивна мова для створення WEB-сторінок, і зараз вже важко уявити собі інтернет без HTML.

Переважна більшість сайтів так чи інакше використовують HTML.
